# **App Name**: CareClarity

## Core Features:

- User Authentication: Secure user authentication with different roles (Parent/Caregiver, Doctor/Consultant, Admin) using email/password and role-based access control.
- AI-Assisted Guidance: Provide AI-assisted guidance based on user queries, retrieving relevant content from a verified knowledge base, ensuring safety through validation and disclaimers. The AI acts as a tool for reasoning over a large dataset to assist parents and caregivers of children with ASD.
- Doctor/Resource Discovery: Match caregivers to doctors/resources based on child age group, condition category, and location or online availability, returning ranked results.
- Profile Management: Allow users to set up and manage profiles with anonymized child-related metadata.  Doctors can manage specialization and availability information.
- Emergency Redirection: Provide an emergency redirection screen for immediate assistance.
- Safety and Compliance: Implement a safety validation layer to detect unsafe inputs, block restricted AI outputs, and trigger emergency guidance when required.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calm and trustworthiness, crucial for a healthcare-related application. Based on the concept of providing a safe haven.
- Background color: Very light blue (#F0F8FF), almost white, maintaining a clean and airy feel to prevent visual clutter. A lighter tint of the primary hue, greatly desaturated.
- Accent color: Muted green (#90EE90), analogous to blue, providing a subtle contrast to highlight important elements without overwhelming the user. (Approximately 30 degrees 'left' of the blue hue.)
- Font pairing: 'PT Sans' for headlines (sans-serif) and 'Literata' for body (serif), offering a balance between modern readability and a comforting, literary feel.
- Use clear, accessible icons that represent different conditions and resources, ensuring ease of navigation for all users.
- Implement a clean and accessible UI design, adhering to accessibility standards to ensure ease of use for parents and caregivers of neurodivergent children.
- Use subtle animations and transitions to enhance user experience, providing feedback on interactions without causing distractions.
